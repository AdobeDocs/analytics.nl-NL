# Machtigingen voor productprofielen voor rapportsuite

Deze lijst maakt een lijst van alle toestemmingenpunten onder de Hulpmiddelen van de Reeks van het Rapport wanneer het vormen van een productprofiel in Adobe Admin Console. Deze toestemmingspunten zijn slechts op de rapportseries van toepassing inbegrepen in dat zelfde productprofiel. Als het huidige productprofiel geen rapportsuites bevat, dan doet het omvatten van deze toestemmingen niets.
