# Opmerkingen bij de Adobe Analytics-release 2026

Leer over de recentste versie voor [ producten van Adobe Experience Cloud ](https://business.adobe.com/products/adobe-experience-cloud-products.html) bijwerkt. Kies voor de nieuwste zelfHelp-documentatie, zelfstudies en cursussen op Experience League.

## Februari 2026 {#feb26}

| Functie en beschrijving | [ Het begin van de Uitvoer ](releases.md) | [ Algemene Beschikbaarheid ](releases.md) |
| ----------- | ------- | ---- |
| **Verbeterde gegevensvoer** <br/> De volgende verbeteringen zijn nu beschikbaar voor gegevensvoer:<ul><li>Verbeterde gebruikerservaring.</li><li>Nieuwe ervaring voor het maken en beheren van kolomsjablonen.</li><li>U kunt nu kiezen wanneer u een kolomsjabloon maakt die u later opnieuw kunt gebruiken in gegevensfeeds. U kunt sjablonen ook verwijderen, bewerken en kopiëren.<br/> eerder, elke gegevensvoer die werd gecreeerd resulteerde in een nieuw kolommalplaatje, dat in een groot aantal ongebruikte kolommalplaatjes en geen manier resulteerde om hen te schrappen of te beheren.</li><li>Toevoegen en filteren op tags.</li><li>Bekijk de geschiedenis van taken voor het doorvoeren van gegevens. (Wanneer de verzoekperiode begon, toen het begon, eindigde, etc.)</li><li>Verstuur of verwerk gegevensfeeds opnieuw. (Vanaf de pagina Taakgeschiedenis)</li><li>Laat aankomen toestaan. U kunt nu gegevens opnemen die zijn ontvangen nadat de gegevensinvoertaak de gegevens heeft verwerkt binnen de ingestelde rapportfrequentie.</li><li>Wanneer u een einddatum voor een gegevensinvoer kiest, wordt de einddatum die u kiest, opgenomen als de laatste dag van de gegevensinvoer.<br/> eerder, werd de einddatum uitgesloten van de gegevensvoer.</li><li>Een exportfrequentie van 15 minuten is nu mogelijk, maar is standaard niet beschikbaar. Om deze optie beschikbaar te maken in uw omgeving, dient u eerst contact op te nemen met de klantenservice van Adobe en te verzoeken dat uw rapportenpakket is geconfigureerd voor ondersteuning van 15-minuten export.</li></ul><p>**Nota:** met deze verbeteringen, worden URLs aan de pagina&#39;s van de gegevensvoer in Adobe Analytics ook bijgewerkt. Werk eventuele bestaande bladwijzers vóór 30 april 2026 bij om naar de nieuwe pagina&#39;s met gegevensfeeds te verwijzen.</p><p>Voor meer informatie, zie [ een gegevensvoer ](/help/export/analytics-data-feed/create-feed.md) creëren en [ leiden gegevensvoer ](/help/export/analytics-data-feed/df-manage-feeds.md).</p> | woensdag 20 januari 2026 | woensdag 24 februari 2026<p>(Oorspronkelijk gepland voor vrijgave op 10 februari 2026)</p> |
| **Nieuwe Adobe Analytics 2.0 de Gids van de Uitgangen API van het Rapport** <p>API&#39;s voor rapportsuite maken, ophalen, bijwerken en verwijderen. Voor meer informatie, zie [ Adobe Analytics 2.0 de verwijzingsgids van de Reeks API van het Rapport ](https://developer.adobe.com/analytics-apis/docs/2.0/apis/report-suites/) en [ Adobe Analytics 2.0 de eindpuntgids van de Reeks API van het Rapport van de Reeks ](https://developer.adobe.com/analytics-apis/docs/2.0/guides/endpoints/report-suites/).</p> | | Februari 2026 |
| **Nieuwe Adobe Analytics 2.0 API Anomaly de Gids van het Rapport van de Opsporing** <p>Maak een geautomatiseerd API-rapport voor Anomaly-detectie. Voor meer informatie, zie de [ Anomaly de eindpuntgids van het Rapport van de Opsporing API ](https://developer.adobe.com/analytics-apis/docs/2.0/guides/endpoints/reports/anomaly/).</p> | | Februari 2026 |
| **Nieuwe Adobe Analytics 2.0 API BasisDatum Trended de Gids van het Rapport** <p>Creeer een geautomatiseerd API Basis Trended KPI Rapport van de Datum van de Datum voor meer informatie, zie {de eindpuntgids van het Rapport van het Rapport van de Detectie API van 0} Anomaly [.](https://developer.adobe.com/analytics-apis/docs/2.0/guides/endpoints/reports/kpi/)</p> | | Februari 2026 |
