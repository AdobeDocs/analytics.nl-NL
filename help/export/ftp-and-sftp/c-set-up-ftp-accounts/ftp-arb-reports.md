# Report Builder geleverde rapporten

Gebruik de Adobe Report Builder add-in voor Microsoft Excel om aangepaste aanvragen te maken van gegevens voor webanalyse, die u in uw Excel-werkbladen kunt invoegen.

Verzoeken kunnen dynamisch verwijzen naar cellen in uw werkblad en u kunt bijwerken en aanpassen hoe de gegevens worden weergegeven. Voor FTP moet u ervoor zorgen dat de host extern beschikbaar is. Voor meer informatie over de [!UICONTROL report builder] geleverde rapporten, zie [Report Builder Help](https://experienceleague.adobe.com/docs/analytics/analyze/report-builder/home.html).

Voor informatie over FTP-beperkingen en gegevensbehoud raadpleegt u [FTP-limieten en gegevensbewaring](/help/export/ftp-and-sftp/ftp-limits.md).
